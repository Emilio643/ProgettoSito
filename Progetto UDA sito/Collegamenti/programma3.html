Eliminazione indice di un'array



Questo programma elimina un indice scelto dall'utente da un'array.
Prima di cominciare vediamo come funziona l'algoritmo. Semplicemente una volta ottenuto dall'utente l'indice eliminato tramite l'utilizzo di un ciclo for, non facciamo altro che sostituire i numeri a partire da quell'indice con il numero affianco. Questa operazione viene ripetuta per n-1 volte dove "n" è la grandezza dell'array.

Cominciamo col dichiarare la grandezza dell'array e l'indice eliminato. Acquisiamo poi i valori che conterrà l'array e l'indice che verrà eliminato da esso.

 int n, indice_eliminato;

  printf("Inserisci grandezza array: ");
  scanf("%d", &n);

 int array[n];

  printf("Inserisci %d numeri:\n");
 for(int i = 0; i < n; i++){

   scanf("%d", &array[i]);

 }

  printf("Quale indice vuoi eliminare? ");
  scanf("%d", &indice_eliminato);

Per comodità stampiamo l'array con gli indici ancora non "manomessi".

 printf("Array con indice ancora intatto: ");
 for(int a = 0; a < n; a++){

   printf("%d ", array[a]);

 }

Adesso applichiamo l'algoritmo descritto precedentemente. Con un ciclo for in cui abbiamo dichiarato una variabile "c" che contiente l'indice da eliminare, settiamo il ciclo in modo che ripeta le istruzioni al suo interno finché "c" è minore di n-1, dove "n" è la grandezza dell'array. Poi sostituiamo il valore dell'indice "c" con il numero affianco. Facendo così abbiamo spostato tutti i numeri a partire dall'indice eliminato di una posizione, e l'indice da eliminare è stato sovrascritto da quello successivo.

 for(int c = indice_eliminato; c < n-1; c++){

  array[c] = array[c+1];

 }

Adesso non ci resta che stampare il nuovo vettore ottenuto.

 printf("Array con indice eliminato: ");
 for(int a = 0; a < n-1; a++){

   printf("%d ", array[a]);

 }

Codice completo:

 #include <stdio.h>

 int main(){

  int n, indice_eliminato;

  printf("Inserisci grandezza array: ");
  scanf("%d", &n);

  int array[n];

  printf("Inserisci %d numeri:\n");
  for(int i = 0; i < n; i++){

    scanf("%d", &array[i]);

  }

  printf("Quale indice vuoi eliminare? ");
  scanf("%d", &indice_eliminato);

  printf("Array con indice ancora intatto: ");
  for(int a = 0; a < n; a++){

    printf("%d ", array[a]);

  }
  printf("\n\n");

  for(int c = indice_eliminato; c < n-1; c++){

   array[c] = array[c+1];

  }

  printf("Array con indice eliminato: ");
  for(int a = 0; a < n-1; a++){

    printf("%d ", array[a]);

  }

 }
