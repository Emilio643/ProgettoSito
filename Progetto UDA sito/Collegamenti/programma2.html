Ordinazione di un array con Bubble Sort



Questo programma ordina il contenuto di un'array con dimensione acquisita da tastiera secondo l'algoritmo del Bubble Sort.

Ma prima di cominciare vediamo un po' come funziona il Bubble Sort.
Secondo Wikipedia:
"In informatica il Bubble sort o ordinamento a bolla è un semplice algoritmo di ordinamento di liste di dati. In esso l'insieme di dati viene scansionato, ogni coppia di elementi adiacenti viene comparata ed i due elementi vengono invertiti di posizione se sono nell'ordine sbagliato. L'algoritmo continua nuovamente a ri-eseguire questi passaggi su tutta la lista fino a quando non vengono più eseguiti scambi, situazione che indica che la lista è ordinata".

Bene dopo aver visto la teoria passiamo alla pratica.

Per prima cosa dichiariamo una variabile che contierrà la grandezza dell'array e acquisiamone il valore tramite scanf(). Una volta fatto ciò creiamo il vettore e tramite un ciclo for acquisiamo i numeri che conterrà l'array.

 int n, temp;

 printf("Inserisci la grandezza dell array: ");
 scanf("%d", &);

 int array[n];

 printf("Inserisci i numeri dell array\n");

 for(int i = 0; i < n; i++){

  scanf("%d", &array[i]);

 }

Adesso mettiamo in pratica quello che abbiamo imparato dalla teoria. Sappiamo che il Bubble Sort va verificare ogni numero singolarmente se è minore del numero affianco. Viene naturale da pensare che avremo bisgono di due cicli innestati in modo che ogni numero possa essere verificato con quello che lo affianca e ripetere questa operazione per il numero della grandezza dell'array. In precedenzza è stata dichiarata una variabile di nome "temp", la utilizzeremo per scambiare i numeri nel vettore.
In pratica sarebbe:

 for(int i = 0; i < n; i++){

  for(int a = 0; a < n-1; a++){

   if(array[a] > array[a+1]){

    temp = array[a];
    array[a] = array[a+1];
    array[a+1] = temp;

   }

  }

 }

Infine con il ciclo for stampiamo il contenuto del vettore.

 printf("L'array ordinato e: ");
 for(int i = 0; i < n; i++){

  printf("%d ", array[i]);

 }

Il codice completo:

 #include <stdio.h>

 int main(){

  int n, temp;

  printf("Inserisci la grandezza dell array: ");
  scanf("%d", &n);

  int array[n];

  printf("Inserisci i numeri dell array\n");

  for(int i = 0; i < n; i++){

   scanf("%d", &array[i]);

  }

  for(int i = 0; i < n; i++){

   for(int a = 0; a < n-1; a++){

    if(array[a] > array[a+1]){

     temp = array[a];
     array[a] = array[a+1];
    array[a+1] = temp;

    }

   }

  }

  printf("L'array ordinato e: ");
  for(int i = 0; i < n; i++){

   printf("%d ", array[i]);

   }

  }
