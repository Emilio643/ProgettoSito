Quante volte un numero si ripete in un 'array


Cominciamo col dichiarare la dimensione dell’array e tramite la funzione scanf() acquisiamo il valore.

 int n;
 printf("Quanti numeri devo controllare? --> ");
 scanf("%d", &n);


Adesso creiamo due array (uno sarà la copia dell’altro), e una variabile che conterà quanti numeri si ripetono.

 int array[n], arraycopia[n], check;



Con un ciclo for acquisiamo i valori del vettore.

 printf("Iserisci %d numeri:\n", n);
 for(int b = 0; b < n; b++){

  scanf("%d", &array[b]);

 }

Con un altro ciclo for settato per eseguire le istruzioni al suo interno finché “i” è minore di “n”, copiamo
il contenuto dell’array in un altro vettore.
N.B. Questo passaggio è facoltativo, poiché la lettera dei numeri ripetuti nell’array
può essere fatta anche confrontando l’array stesso evitando così l’utilizzo di una copia.

 for(int i = 0; i < n; i++){

  arraycopia[i] = array[i];

 }

Adesso andiamo ad effettuare finalmente la lettura dell’array. Creiamo due cicli innestati uno per scorrere l’indice
del primo array, l’altro per scorrere l’indice dell’array copia. Tramite if verifichiamo se il numero di indice “a”
dell’array è uguale al numero di indice “b” dell’array copia, se la condizione è vera, “check” viene incrementato di 1 unità.
Viene poi effettuata un'altra verifica, se stiamo verificando lo stesso indice il valore di “check” che si sarà dovuto
incrementare per forza verrà annullato visto che è andato a verificare un numero uguale.

 for(int a = 0; a < n; a++){

  for(int b = 0; b < n; b++){

   if(array[a] == arraycopia[b]){

    check++;

  }

   //se stiamo verificando lo stesso indice annulliamo la conta di prima
   if(a == b){

     check--;

   }
  }
 }

Infine stampiamo il valore della variabile check che conterrà il numero di elementi duplicati.

 printf("Il numero di elementi duplicati e: %d", check);

Codice Completo:

 #include <stdio.h>

 int main(){

   int n;

   printf("Quanti numeri devo controllare? --> ");
   scanf("%d", &n);

   int array[n], arraycopia[n], check;

   printf("Iserisci %d numeri:\n", n);
   for(int b = 0; b < n; b++){

    scanf("%d", &array[b]);

   }

   //********copia array*********
   for(int i = 0; i < n; i++){

    arraycopia[i] = array[i];

   }

   //verifico gli elementi duplicati confrontando i due array

   for(int a = 0; a < n; a++){

    for(int b = 0; b < n; b++){

     if(array[a] == arraycopia[b]){

      check++;

     }

    // se stiamo verificando lo stesso indice annulliamo la conta di prima
    if(a == b){

     check--;

    }

   }

  }

 printf("Il numero di elementi duplicati e: %d", check/2);

 }
